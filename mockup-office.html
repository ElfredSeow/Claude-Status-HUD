Design spec — what each element represents
Trigger
🖱️ Hover the traffic-light overlay to peek at the office (auto-closes when mouse leaves)
🖱️ Click to pin it open — click again or press Esc to dismiss
📌 A pinned popup stays on top even while you click other windows
Office floor layout
🏢 One cubicle per live Claude session; scrolls if > 6
📂 Cubicle title = the project folder name (cwd basename)
🌱 Tiny decor (plant, window, coffee) makes each cell feel alive
Claude robot states
🟢 Idle — gently bobbing, coffee in hand, green glow; bubble: "Ready!"
🟡 Busy — arms typing rapidly, amber glow; bubble shows the active tool
🔴 Needs approval — pulsing, exclamation sign, red glow; bubble: "Please approve!"
⚫ None / vacant — dimmed robot, grey desk, empty bubble
Data shown per session
💬 Chat bubble: current tool name (e.g. "Bash: npm test") or idle message
🖥️ Mini monitor: colour-coded code lines showing activity
🏷️ Name plate: project name + status pill (Idle / Busy / Needs approval)
📡 Antenna glow colour matches state; pulses when busy/permission
